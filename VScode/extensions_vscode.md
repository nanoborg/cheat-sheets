## VS code setup 

* Install shell command through the command palette
    allows to open vs code through the terminal. 
    1. `cmd + shift + p`
    2. `code`
    3. `Shell Command: Install 'code' command in PATH`

* Open current directory in terminal
    by typing ``` code .```

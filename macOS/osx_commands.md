## | Global shortcuts | 
---

### Delete whole line:
    cmd + backspace

### Delete single word:
    option + backspace

### Cycle ***forwards*** between active applications:
    cmd + tab

### Cycle ***backwards*** between active applications:

1.      cmd + tab

2.      cmd + `

### Switch between windows of the same application:
    cmd + `

### Spotlight: 
    cmd + spacebar

### Close window:
    cmd + w

### Quit appication:
    cmd + q

### Seach in app:
    cmd + f

### Cycle through search:
    cmd + g

### Open display settings:
    option + f2 (+ brightness)

### Jump to ***end*** of current line:
    cmd + right arrow

### Jump to ***start*** of current line:
    cmd + left arrow

### Jump to ***start*** of page:
    cmd + up arrow

### Jump to ***end*** of page:
    cmd + down arrow

### Scroll ***down***:
    down arrow

### Scroll ***up***:
    up arrow
    
### Custom screenshot:
    cmd + shift + 4

### Hide application:

    cmd + h
